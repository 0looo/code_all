# 단축키 모음 (기본단축키, 사용자 설정 X)
행,렬 추가
    ctrl + '+'
        : 4가지 옵션 중 선택(행, 렬, 해당칸 밀기 등)
행, 렬 삭제
    ctrl + '-'
        :4가지 옵션 중 선택(행, 렬, 해당칸 당기기 등)
        
필터 적용
   Alt + 'a' + 't'

# 함수
INDEX(데이터셋, INDEX값, COL값)
    : INDEX값과 COL값을 숫자로 넣으면 데이터셋을 기준으로 해당 값을 출력

MATCH(찾는 값, 찾는 범위, #)
    : 찾는 범위 안에서 찾는 값에 해당하는 순서를 출력
        #
            0 : 정확히 일치할 경우
            1 : 작은 값들 출력
            -1 : 큰 값을 출력
